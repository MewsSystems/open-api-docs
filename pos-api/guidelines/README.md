# Usage guidelines

This section provides important guidelines on how to use the **Mews POS API**. For a deeper understanding of **Mews** concepts, see [Concepts](../concepts/). For information about individual endpoints or operations, see [API Operations](/broken/pages/9cAxzDNLVliB7LXfQj2c).

> #### Terminology
>
> Some of the terms used in the API, such as _enterprise_, _customer_ or _resource_, may not be familiar. For a full description of all the terms used, see the [Mews Glossary for Open API users](https://help.mews.com/s/article/Mews-Glossary-for-Open-API-users?language=en_US).
